# Privacy Policy

Our commitment to privacy is simple: **we don't collect, store, or sell your personal data.**

This website serves as a documentation and demonstration hub for the amberbockel/ui design system. 

## Data Collection

- **Analytics:** We may use basic, privacy-friendly analytics (if any) to understand general website traffic and usage patterns (e.g., page views). These tools do not track individual users across the web or collect personally identifiable information (PII).
- **Cookies & Local Storage:** We do not use tracking cookies. We only use your browser's local storage to remember your theme preference (Light or Dark mode).

## Open Source

The source code for this design system is fully open source and hosted on GitHub. You are free to inspect the code to verify that no hidden tracking or data collection mechanisms are present.

## Contact

If you have any questions about this privacy policy, please refer to the project repository on GitHub.
